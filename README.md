# Predictive Analytics Using Historical Data

## Project Overview
This project uses historical sales data to build a predictive analytics model for forecasting future sales trends using Linear Regression.

The project includes:
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Predictive modeling using Linear Regression
- Future sales forecasting
- Model accuracy evaluation
- Data visualization

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset
The project uses the Superstore Sales dataset containing historical sales records including:
- Order Date
- Sales
- Profit
- Category
- Region

---

## Project Workflow

1. Load historical sales dataset
2. Clean and preprocess data
3. Aggregate monthly sales trends
4. Train Linear Regression model
5. Predict future sales trends
6. Evaluate model accuracy using:
   - R² Score
   - Mean Absolute Error (MAE)
7. Visualize actual and predicted sales trends

---

## Model Accuracy
The model performance was evaluated using:
- R² Score
- Mean Absolute Error (MAE)

---

## Forecasting
The model predicts future sales trends for the next 6 months based on historical sales patterns.

---

## Output Visualizations
- Actual vs Predicted Sales Trend
- Future Sales Forecast Graph

---

## Conclusion
This project successfully demonstrates predictive analytics using historical sales data. Linear Regression was used to forecast future sales trends and evaluate prediction performance.
